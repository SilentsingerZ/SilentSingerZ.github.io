Introduction

TEllMe is an application in iOS 11 platform that based on the Speech Kit. This lifestyle application aims to aid ticket purchasing and provide convenience for disabled people.
TEllMe would aid users purchasing tickets for concerts and competitions. The speech recognition feature allows users to search and to purchase without looking at the screen, as well as aiding users who have a vision-related disability. Also, the location services enable the users to purchase ticket nearby. TEllMe could search the tickets based on the voice input and filter the results through comparing the prices, discount, and location. TEllME could provides convenience for disabled people while guarantee the disabled access at the same time.

This documentation is to help you regarding all the features of TEllMe. Please go through the documentation carefully to understand how to use this application and what you would expect.

Requirements

You will need an iPhone to use this application

  • Software must updates to iOS 10.1 or higher
  • TEllME only supports iOS currently, the android version is not available.

    • Installing Instruction #back to top

    To install TELlMe, please unzip the zip archive TELlMe first.

    Open the xcode file SearchVUI.xcodeproj

    In the Xcode, lick the upper left corner and choose your desire platform.

    Run the code and the application will appear on the platform you choose.
    Have fun!

    Getting Started #back to top

    Basic Interface #back to top

    A lunch screen will display once you open the application.

    Then, you will see our basic interface including Speical Needs Organization and Great Concerts.

    Tapping the artist name, you will get information regarding their concerns.
    The page will show the available date, location, time schedule, and concert information.

    Choosing your preferred time and location, the application will lead you to the purchse website.

    Voice Recognition #back to top

    TELlMe also supports voice input for searching and purchasing.

    By tapping the right upper concern, you will be able to speak whatever you want to search.

    The searching inferface will display your voice input. You could search multiple artists at the same time.

    The inferface will display the searching results. Tapping the artist name to get full information.

    You could also search the concert information through location.

    The searching results will display all the available concerts information based on your desired theatre.

    You could also search through artist and your preferred theatre.

    Again, the interface will showing all the possible results.
