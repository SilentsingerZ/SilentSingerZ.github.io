About this app

Instructions for using the app

  • This is a 1-key interface.
  • Users could only wait selector go to next variable one by one.
  • Finally press 'Enter' to input selected variable with a short time period.

Description of test

  • Use two parts of my body: 1. Finger. 2. Nose.
  • Enter the equation '71 * 39 ='.
  • I only did one trial instead of five for each body parts and compute the average time.

Description of result

  • By using my finger, it takes me 65.04 seconds to type in the formula since I actually can not control anything.
  • By using my nose, it takes me 192.34 seconds to type in the formula. Many errors made this time, since I can't see the screen by using my nose
  • Some errors inevitably happened since I can't see the screen by using my nose. When I miss one variable, I need to wait about 15 seconds for another round.
